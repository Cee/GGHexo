# 校对和定稿说明

## 校对

校对的目的：**修改内容**，在**内容正确**的前提下，提高内容**可读性**。

常见问题：

- 长句、难句，需要适当截断，调整长度
- 英文直译，需要调整语句顺序
- 表述过于书面化，需要修改为适合阅读的类型，更加口语化

## 定稿

定稿的目的：**统一内容格式，保证发布不出问题**，同时**再次校对文章内容**。

常见的格式错误：

- 文章头部格式错误
- 没有加`<!--more-->`标签
- 中英文数字混排没加空格
- 英文大小写错误（比如 iOS、GitHub）
- Markdown 格式错误（比如每段之间、段落和代码之间、段落和标题之间没加空行，标题的`##`和内容之间没加空格）
